# CD-Collection
A java application to create and manage **Music Albums**.
The current version manages the following entities:
  * Artists;
  * Artist photos;
  * Albums;
  * Album cover arts;
  * Genre;
  * Songs;
  * Record companies;
  
Technologies Used:
  * Java Core;
  * Java persistence Api;
  * Hibernate 5.x;
  * MySQL database;
  
This project was done for learning purposes.
